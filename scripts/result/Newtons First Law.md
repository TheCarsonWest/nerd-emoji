# Newton's First Law (The Law of Inertia)

Newton’s First Law is the foundational principle of classical mechanics. It establishes the relationship between the motion of an object and the forces acting upon it.

## The Definition
Newton's First Law states: 
> "An object at rest stays at rest, and an object in motion stays in motion with the same speed and in the same direction unless acted upon by an unbalanced force."

Mathematically, this means that if the net external force acting on an object is zero, the object's acceleration is also zero:
$$\sum \vec{F} = 0 \iff \vec{a} = 0$$

## Inertia and Mass
The property of an object to resist changes in its state of motion is called **inertia**. Inertia is not a force; it is an inherent property of matter. The measure of an object's inertia is its mass ($m$).

| Property | Description | Relationship to Motion |
| :--- | :--- | :--- |
| **Inertia** | Resistance to changes in velocity. | Higher inertia means more force is needed to change velocity. |
| **Mass ($m$)** | The quantitative measure of inertia. | Measured in kilograms ($kg$). |

An object with larger mass will require a greater net force to achieve the same acceleration as an object with smaller mass.

## Equilibrium
When the net force acting on an object is zero ($\sum \vec{F} = 0$), the object is said to be in a state of **Equilibrium**. It is important to distinguish between two types of equilibrium:

1.  **Static Equilibrium:** The object is at rest ($v = 0$ and $a = 0$).
2.  **Dynamic Equilibrium:** The object is moving at a constant velocity ($v = \text{constant}$ and $a = 0$).

In both cases, the object is not accelerating. If you observe an object moving in a straight line at a constant speed, you must conclude that the net force acting on it is zero, even if individual forces (like [[Kinetic and Static Friction]] or applied forces) are present.

## Applications in AP Physics C
In AP Physics C, you will frequently apply the First Law when analyzing systems. Often, problems will state that a system is "moving at constant velocity" or "hanging in equilibrium." These phrases are code for $\sum \vec{F} = 0$.

### Steps for Solving First Law Problems:
1.  **Identify the object:** Isolate the body of interest.
2.  **Draw a Free-Body Diagram:** Account for all forces acting *on* the object (gravity, normal force, [[Spring Forces]], tension, etc.). See [[Forces and Free-Body Diagrams]].
3.  **Set up the Equations:** Break forces into $x$ and $y$ components.
    *   $$\sum F_x = 0$$
    *   $$\sum F_y = 0$$
4.  **Solve:** Use the equations to find unknown variables like tension or the coefficient of friction.

By mastering the First Law, you establish the baseline for understanding how systems behave before you introduce the more complex dynamics found in [[Newtons Second Law]].
