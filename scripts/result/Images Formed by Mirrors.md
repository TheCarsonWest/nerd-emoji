# Topic 13.2: Images Formed by Mirrors

When light undergoes [[Reflection]], it forms images based on the geometry of the reflecting surface. Mirrors can be classified as plane (flat), concave (converging), or convex (diverging).

## The Mirror Equation and Magnification

The relationship between the object distance ($d_o$), the image distance ($d_i$), and the focal length ($f$) is defined by the mirror equation:

$$\frac{1}{f} = \frac{1}{d_o} + \frac{1}{d_i}$$

The magnification ($m$) of an image, which determines whether it is upright or inverted and its relative size, is given by:

$$m = \frac{h_i}{h_o} = -\frac{d_i}{d_o}$$

Where:
*   $h_i$ is the image height.
*   $h_o$ is the object height.

## Sign Conventions

Correct usage of signs is critical for determining image characteristics. Use the following convention for mirrors:

| Variable | Positive (+) | Negative (-) |
| :--- | :--- | :--- |
| **Focal Length ($f$)** | Concave mirror (converging) | Convex mirror (diverging) |
| **Object Distance ($d_o$)** | Object is in front of mirror | Object is behind mirror |
| **Image Distance ($d_i$)** | Real image (in front of mirror) | Virtual image (behind mirror) |
| **Magnification ($m$)** | Upright image | Inverted image |

## Types of Mirrors

### Plane Mirrors
Plane mirrors are flat surfaces. They always produce virtual, upright images that are the same size as the object. For a plane mirror, the focal length is infinite ($f = \infty$), resulting in $d_o = -d_i$.

### Spherical Mirrors
Spherical mirrors are sections of a sphere. The focal length is half the radius of curvature ($R$):

$$f = \frac{R}{2}$$

*   **Concave Mirrors:** These curve inward. They can form real or virtual images depending on where the object is placed relative to the focal point. If the object is placed outside the focal point ($d_o > f$), the image is real and inverted. If the object is placed inside the focal point ($d_o < f$), the image is virtual, upright, and magnified.
*   **Convex Mirrors:** These curve outward. They always produce images that are virtual, upright, and reduced in size, regardless of object position.

## Ray Tracing
To visually determine image location, one can use three standard principal rays:
1.  **Ray 1:** Travels parallel to the principal axis and reflects through the focal point.
2.  **Ray 2:** Travels through the focal point and reflects parallel to the principal axis.
3.  **Ray 3:** Travels through the center of curvature ($C = 2f$) and reflects back along the same path.
